Library

Use my algorithm to correctly obtain other processes' arguments

Fast, and correct algorithm

libgetargv uses the same code as getargv, so it is just as fast and resilient as the standalone tool.

Multi lanugage

  • Libgetargv is written in C, out of necessity
  • The header for libgetargv is C++ compatible
  • I provide a safe Rust crate that wraps the bindings to the lib
  • I provide Swift bindings for the lib
  • more language bindings are on the way
